The R+L Carriers Shipping Plugin adds your R+L Carriers shipping rates to WooCommerce cart and checkout pages.

== Description ==

This plugin has been republished by R+L Carriers. You can download the new version [here](https://wordpress.org/plugins/woo-shipping-method-rl-carriers/).

== Installation ==

= Minimum Requirements =

* WordPress v3.8
* WooCommerce v2.0.18
* PHP version 5.2.4 or greater
* MySQL version 5.0 or greater
* An R+L Carriers MyRLC account and B2B API Key

= Quick Install =
